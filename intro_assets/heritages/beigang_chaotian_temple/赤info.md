# 簡介
赤崁樓的身世超級波折，它完美轉譯了台灣三百多年來的政權更迭。最早是 1653 年荷蘭人蓋的「普羅民遮城」，作為當時的行政與商業中心；後來鄭成功拿下台灣，把它改成最高行政機關。

到了清朝，原本的荷蘭城堡因為地震和戰亂壞得差不多了，清朝官員乾脆在荷蘭人的紅磚地基上，蓋起了傳統的中國式樓閣（海神廟與文昌閣）。所以你現在走進去，等於是一口氣踩在三個不同時代的歷史斷層上，空間的層次感非常豐富！

# 建築特色
因為上面的歷史背景，赤崁樓的建築本身就是個超強的「混搭風」。最核心的視覺焦點有兩個：

荷蘭時代的原始入口：在海神廟的基座底下，還留著一道荷蘭時期的磚砌古井和入口遺跡。站在那裡摸摸那些厚實的紅磚，真的會有穿越時空的感覺。

九座「贔屭（ㄅㄧˋ ㄒㄧˋ）」御龜碑：海神廟前面那一整排、由長得像烏龜的龍之子「贔屭」馱著的石碑。那是清朝乾隆皇帝為了表彰平定林爽文事件賜下的。最好玩的是，當年本來運了十隻過來，結果有一隻在搬運時不小心掉進海裡，後來被打撈起來放在台南的保安宮，所以赤崁樓這裡永遠「缺一角」，只有九隻。

# 神明與特色
赤崁樓不只是看古蹟，裡面的「文昌閣」更是台南考生的精神寄託。二樓供奉著掌管文運的魁星爺，祂的造型超級生動——右手拿硃砂筆、左手拿墨斗，一腳踩著鰲魚（代表獨占鰲頭），另一腳往後踢著星斗。
