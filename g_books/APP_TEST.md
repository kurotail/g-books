## 可測試流程

### A. 學生登入與初次設定
1. 登入頁輸入組長姓名 + 座號（例：`王小明` / `1`）。
2. 初次登入 → **小組頭像**（上傳 / 拍照 / 可略過）→ **小組命名** → **小組總覽**。
3. 小組總覽：列出組員卡片；點任一組員可上傳頭像（裁切 → 確定 → 自動回總覽）。
   按「進入遊戲」才**完成註冊**並進入古蹟選擇。流程每步皆可返回、已填資料保留。

> mock 重開 App 會重置 `hasCompletedSetup`，可重複測試初次設定流程。

### B. 古蹟檢視與遊玩（學生）
1. **古蹟選擇**：左右滑動卡片；點「古蹟資訊」開 InfoDialog（左側分頁 + 文圖文）。
   北港按「出征」進入。
2. **古蹟檢視**：左上面板（常駐顯示組名、向下摺疊）含 古蹟資訊 / 編輯古蹟 / 小組資訊 / 登出。
3. **編輯古蹟**：底部「原料庫」拖曳原料到 slot；點已放置原料可拆下放回背包。
   點「原料庫」看圖鑑，點原料看 InfoDialog（左卡圖 + 右文圖文）。

### C. 管理者後台（編輯古蹟設定）
1. 登入頁左下「教師 / 管理者登入」→ 輸入 `admin` / `admin123`。
2. `/admin` 選古蹟 → 進編輯器，三種模式：
   - **Slot**：點空白新增、拖曳移動、八方控制點縮放、兩指縮放畫布。
   - **原料對應**：選一個原料 → 點 slot 切換可放（綠 = 可放）。
   - **物品**：列出該古蹟原料圖，設定**名稱**與**等級**(1/2/3)。
3. 按「**儲存**」→ 寫回假後端並即時生效。重抓鈕（↺）可捨棄未存變更、重新向伺服器取設定。
4. **驗證單一來源**：改北港某原料名稱/等級並儲存 → 重開 App → 管理者再看為新值，
   且**學生端原料庫**也顯示新值。

### D. 教師
- `teacher` / `teacher123` 登入 → `/teacher` 占位頁（「功能開發中」+ 登出）。

## 重置測試資料
- **學生 / 背包**：重開 App 即重置（記憶體）。
- **古蹟設定**：刪除 App 文件區的 `gb_admin/` 資料夾 → 下次啟動回到 assets 種子值。
